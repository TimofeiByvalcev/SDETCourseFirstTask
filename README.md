# SDETCourseFirstTask
This is a project for the first task in a SDET course

## Technology stack

1. Java

2. Maven

3. Selenium

4. Allure

5. Junit 4

6. JavaFaker


## Test execution

For test execution need to run this command in terminal:

```powershell
    
    mvn clean test
    
 ```

For generating and view allure report run:

```powershell
    
    mvn allure:serve
    
```
